# AI Prompt Enhancer Pro ✨
A Streamlit web application that takes basic ideas and engineers them into highly structured prompts for Large Language Models.

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
## 🔑 Getting your Free API Key
This application is powered by Google's Gemini 2.5 Flash model, which includes a generous free tier for developers. 

1. Go to [Google AI Studio](https://aistudio.google.com/).
2. Log in with your Google account.
3. Click **Get API Key** on the left menu to generate your key.

**🛡️ Security & Privacy Note:** Your API key is perfectly safe. Because this app runs entirely locally on your own machine, your key is only used to connect directly to Google's servers. It is never saved, logged, or sent anywhere else.
